# Ali Moto Workshop

Official repository for the Ali Motorcycle Workshop website.

## Overview
Ali Moto Workshop is a responsive, single-page website for a motorcycle workshop in Kathmandu, Nepal. It presents workshop services, contact details, and a visual gallery in a fast, lightweight static site format.

## Highlights
- Mobile-friendly single-page layout
- Service-focused content for local customers
- Gallery section for workshop visuals
- SEO-ready metadata in the main HTML page
- No build tools required (pure HTML/CSS/JS)

## Project Structure
- `index.html` - Main website page
- `gallery/` - Image assets used across the site

## Run Locally
1. Clone this repository.
2. Open `index.html` directly in your browser.

Optional (recommended): serve with any static server for local testing, for example VS Code Live Server.

## Deployment
This project can be deployed to any static hosting platform, including:
- GitHub Pages
- Netlify
- Vercel
